# 💤 Yapay Zeka Destekli Uyku Takip Sistemi ("Uyan Yeğen")

Bu proje, bilgisayar kamerası üzerinden gerçek zamanlı yüz hatlarını tarayarak göz açıklığı oranını (EAR - Eye Aspect Ratio) hesaplar. Kullanıcının uykuya daldığını tespit ettiğinde sesli bir alarm devreye girerek uyarı verir. 

## 🚀 Kullanılan Teknolojiler
* **Python** (Ana Programlama Dili)
* **OpenCV** (Görüntü İşleme ve Kamera Kontrolü)
* **MediaPipe** (Yüz Hatları ve Göz Tespiti)
* **Pygame** (Sesli Uyarı Sistemi)
